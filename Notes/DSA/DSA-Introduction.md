## üìå What is `DSA`?
**DSA (Data Structures and Algorithms)** is the combination of:
- **Data Structures** ‚Üí Ways of organizing and storing data.
- **Algorithms** ‚Üí Step-by-step procedures to solve problems efficiently.

> So, DSA is an efficient way to store and organize data in a step by step process.

Together, DSA helps you write programs that are :
- Fast  
- Optimized  
- Scalable  
- Easy to maintain

### Examples:
- **Array**
- **Linked List**
- **Stack**
- **Queue**
- **Tree**
- **Graph**
- **Hash Table**

### Example (Array)
```python

```

---

## ‚öôÔ∏è What are Algorithms?
- Algorithms = Step-by-step methods to solve a problem.

### Examples:

- Searching
- Sorting
- Recursion
- Graph traversal (DFS, BFS)

### Example (Simple algorithm)
```python
def find_max(arr):
    max_val = arr[0]
    for x in arr:
        if x > max_val:
            max_val = x
    return max_val
```
