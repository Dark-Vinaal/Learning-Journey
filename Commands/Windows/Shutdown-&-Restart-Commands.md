## 🔌 Shutdown & Restart Commands

### 📌 `shutdown /f /r /fw /t 0`

#### Description:

- Force restart into firmware (UEFI/BIOS).
```cmd
shutdown /r /fw /f /t 0
```

#### Where & When Used:

- BIOS/UEFI access

---

### 📌 `shutdown /f /r /t 0`

#### Description:

- Immediate forced restart.
```cmd
shutdown /f /r /t 0
```

#### Where & When Used:

- System reboot

---
