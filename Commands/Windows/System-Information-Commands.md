## 🖥️ System Information Commands

### 📌 `systeminfo`

#### Description:

- Displays detailed system information.
```cmd
systeminfo
```

#### Where & When Used:

- System auditing
- Hardware & OS details

---

### 📌 `sysinfo64`

#### Description:

- Displays system info (if Sysinternals installed).
```cmd
sysinfo64
```

#### Where & When Used:

- Advanced system diagnostics

---
